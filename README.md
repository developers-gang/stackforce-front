# Stack Force Frontend Application
  stackforce.ru
## Installation
1. `yarn install` (or `npm install` for npm)
## Usage
**Development**

`yarn start`

* Build app continuously 
* App served @ `http://localhost:3000`

**Build**

`yarn build`

* Build app 
---

**Note**: replace `yarn` with `npm` in `package.json` if you use npm.
